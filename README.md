# Enterprise AI Toolkit

AI 助手企业应用的实用项目集合。  
聚焦多 Agent 协同、FastAPI 接口与 Docker 部署，为企业自动化场景提供可落地的 MVP。

---

## 项目列表

### 1. AI Digital Employee MVP（`ai_digital_employee_mvp`）
一个完整可运行的 AI 数字员工最小可行产品。

- **多 Agent 协同**：销售 Agent、技术 Agent、报价 Agent、跟进 Agent
- **客户消息自动分析**
- **FastAPI 后端接口**
- **OpenAI 调用示例**
- **PostgreSQL 数据存储**
- **Redis 任务队列**
- **Docker 一键部署**

> 详细使用方法请查看项目内的 README。
